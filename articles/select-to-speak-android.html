Setup & Use Select to Speak on Android

Learn how to setup and use select to speak on your Android device.


Setup Select to Speak

Setup a shortcut to use Select to Speak and adjust speech settings such as the reading voice or enable experimental features.


Google Pixel

  1. Open Settings
  2. Scroll down and tap Accessibility
  3. Tap Select to Speak
  4. Enable Select to Speak Shortcut
  5. Select a shortcut to enable and use Select to Speak: Accessibility button, Volume keys, or Accessibility gesture (if swipe navigatiion is in use)
  6. Tap Settings to edit more settings:
    ‣ Text to Speech Settings: Adjust the system speech language, rate, and pitch.
    ‣ Change Reading Voice: Change the voice Select to Speech uses.
    ‣ Read in Background (Experimental Feature): Select to Speak will continue reading when youo switch between apps.
    ‣ Read Text on Images (Experimental Feature): Tap on text within photos or the caera viewfinder to hear text read aloud.

Samsung Galaxy

  1. Open Settings
  2. Scroll down and tap Accessibility
  3. Tap Installed Services
  4. Tap Select to Speak
  5. Enable Select to Speak Shortcut
  6. Select a shortcut to enable and use Select to Speak: Accessibility button, Volume keys, or Accessibility gesture (if swipe navigatiion is in use)
  7. Tap Settings to edit more settings:
    ‣ Text to Speech Settings: Adjust the system speech language, rate, and pitch.
    ‣ Change Reading Voice: Change the voice Select to Speech uses.
    ‣ Read in Background (Experimental Feature): Select to Speak will continue reading when youo switch between apps.
    ‣ Read Text on Images (Experimental Feature): Tap on text within photos or the caera viewfinder to hear text read aloud.

How to Use

When Select to Speak is enabled, you can do any of the following:

  • Tap an item to hear it spoken aloud.
  • Drag your finger accros the screen to select multple items.
  • Tap Play to hear everything read aloud.
  • Tap text while using the camera to hear text read aloud in the live camera feed.
